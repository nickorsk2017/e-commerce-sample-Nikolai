## Environment tested

node v18.12.1

yarn 1.22.19

## Instaling

```bash
cd ./front
yarn install
```

```bash
cd ./mock-server
yarn install
```

## Running

1. Run mock server

```bash
cd ./mock-server
yarn start
```

2. Run nextjs app

```bash
cd ./front
yarn dev
```

Open to see app [http://localhost:4000](http://localhost:4000)
